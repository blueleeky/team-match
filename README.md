# 배드민턴 팀 매치 기록

서버 없이 브라우저에서 바로 사용하는 정적 웹앱입니다.

## 실행

`index.html` 파일을 브라우저로 열면 됩니다.

## 주요 기능

- A팀/B팀 팀명 변경
- 선수 이름 관리
- 대진표 기준 경기 일정 표시
- 1코트, 2코트, 단체전 필터
- 경기별 A팀/B팀 점수 입력
- 높은 점수를 받은 팀 자동 승리 처리
- 승리 팀 승점 1점 자동 반영
- 개인 승리 순위 자동 계산
- 단체전은 팀 승점만 반영하고 개인 기록에서는 제외
- 브라우저 localStorage 저장
- JSON 백업/복원
- 전체 초기화

## 데이터 관리 방식

경기 일정은 선수 이름 문자열이 아니라 선수 ID를 참조합니다. 선수 이름을 수정하면 일정표와 랭킹에 표시되는 이름이 함께 변경됩니다.
